## Next session target

**Bring up the app locally on the Mac.** Concrete steps:

1. Install Bun if not already: `curl -fsSL https://bun.sh/install | bash`.
2. `bun install` — should be fast since `bun.lock` is already committed and pinned (20s cold from empty node_modules, near-instant on a warm cache).
3. `cd src-tauri && cargo check --locked` — confirm the Rust tree builds against the Tauri 2 ecosystem at pinned versions.
4. `bun run tauri:dev` — confirm the webview renders the placeholder and `invoke("altr_version")` returns something.
5. Add a throwaway icon set (any 1024×1024 PNG through `tauri icon`) so `tauri build` doesn't fail later.
6. First commit: `chore(scaffold): initial Tauri 2 + React scaffold, migration 0001, CI`.

Then Week 1 work begins: the graph store in Rust (`src-tauri/src/graph/`) — typed wrappers over the `nodes` table, a tokio broadcast channel for events, and the first real `#[tauri::command]` (`graph_create_project`).
